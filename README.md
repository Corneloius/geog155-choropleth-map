# GEOG 155 — Peptide & Wellness Interest Choropleth Map

Interactive U.S. choropleth map showing regional peptide supplement search interest.

## How to View
Download `choropleth-map.html` and open it in any browser — no server needed.

## Assignment
GEOG 155 | University of Nebraska-Lincoln | Spring 2026  
Stage 2: Map Creation & Documentation  
Due: 05/01/2026

## Data Sources
- Google Trends Search Interest Index (2024)
- CDC NHANES Dietary Supplement Data (2021)
- Council for Responsible Nutrition Consumer Survey

## Map Type
Choropleth — regions colored by quantitative intensity using a single-hue navy-to-light-blue progression (Cromley & Cromley, 2009).

## Tech Stack
HTML + Leaflet.js + Vanilla JS — no build tools required.
